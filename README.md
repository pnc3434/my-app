Описание приложения
Назначение и основные возможности
Разработано React-приложение "Еженедельные задачи" - интерактивный планировщик задач с возможностью управления расписанием на разные недели.

Ключевые функции:

Отображение дней недели с соответствующими датами

Добавление новых задач для каждого дня

Удаление выполненных задач

Навигация между неделями (предыдущая, текущая, следующая)

Автоматическое сохранение данных в localStorage

Адаптивный дизайн для разных устройств

Особенности реализации
Технический стек:

React с функциональными компонентами


Локальное хранилище браузера (localStorage)

CSS Grid для адаптивной верстки

Архитектурные решения:

Хранение данных: Задачи привязаны к конкретным датам, что позволяет сохранять их при переходе между неделями

Управление состоянием: Использование отдельных состояний для задач и вводов данных

Персистентность: Автоматическое сохранение в localStorage с восстановлением при загрузке

Компоненты:

Главный компонент App содержит всю логику приложения

Динамическое отображение дней недели с датами

Интерактивные элементы управления (кнопки, поля ввода)

Реализованный функционал
Обязательные требования:

Работа с компонентами React

Управление состоянием через useState

Обработка событий (onClick, onChange)

Хранение данных в localStorage

Дополнительные возможности:

Навигация между неделями

Привязка задач к конкретным датам

Автосохранение данных

Адаптивный дизайн

Поддержка клавиши Enter для добавления задач
